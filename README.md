# Fiber Bootstrap
Fiber bootstrap for rapid development using Go-Fiber / Gorm / Validator.

# Components
* Fiber
  * Html Engine Template
  * Logger
  * Monitoring
* Gorm
  * PGSQL Driver
* Validator  
* Env File

# Router 
API Router `/api` with rate limiter middleware  
Http Router `/` with CORS and CSRF middleware
